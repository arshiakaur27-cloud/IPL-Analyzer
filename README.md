# IPL Player Performance Analyzer 🏏

## 📌 Overview

This project analyzes player performance in the Indian Premier League (IPL) using Python.

## 🎯 Features

* Calculates total runs, balls faced, and strike rate
* Filters players with at least 100 balls
* Identifies top performers
* Visualizes data using graphs

## 🛠️ Tech Stack

* Python
* Pandas
* Matplotlib

## 📂 Project Structure

│── data/

## 🚀 How to Run

1. Install dependencies
2. Run `analysis.py`
